#### 内存分配对齐
内存分配，iOS中内存分配都是以16的倍数来分配的，为了提高访问速度。
一个指针8个字节，一个int4个字节，一个double8个字节。最终分配的内存，都是**按16的倍数**来分配的。
```
@interface Person : NSObject
{
    int _age;
    int _no;
}
@end

Person *p = [Person new]; 
提问P对象占用多少内存？
```
编译成结构体后
```
struct Person_IMPL{
    Class isa;	//指针8字节
    int _age;	//整形4字节
    int _no;	//整形4字节
};
```
